Module: Building

The gulp tasks file. It allow to run some tasks from command line interface.
The available tasks are:
  • help
  • patch
  • clean
  • lint
  • doc
  • unit
  • bench
  • test
  • build-test
  • build
  • release
You could find a complet explanation about these tasks using: npm run help.
